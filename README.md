# Simpsons Quotes Battle

Simpsons Quotes Battle es un juego web interactivo y bilingüe donde los jugadores deben adivinar qué personaje de Springfield dijo una cita famosa. El proyecto incluye múltiples modos de juego, puntuación, modo oscuro, efectos de sonido y un diseño visual retro inspirado en la estética de Los Simpson.

## Características

- Modo clásico, supervivencia y contra reloj
- Interfaz bilingüe en español e inglés
- Modo oscuro y tema claro
- Sistema de puntuación y récords locales
- Efectos de sonido y confeti de celebración
- Diseño responsive para desktop y móviles

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript vanilla
- Bootstrap 5
- Bootstrap Icons
- Google Fonts
- Canvas Confetti

## Cómo ejecutar el proyecto

1. Clona este repositorio:
   ```bash
   git clone https://github.com/francisco-hernandez/SimpsonsQuotesBattle.git
   ```
2. Entra al directorio del proyecto:
   ```bash
   cd SimpsonsQuotesBattle
   ```
3. Inicia un servidor local:
   ```bash
   python3 -m http.server 8080
   ```
4. Abre en tu navegador:
   ```text
   http://localhost:8080
   ```
